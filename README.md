# Nyaya AI

Nyaya AI is an AI-driven legal assistance platform designed to help users explore and understand Indian legal cases through semantic search and contextual responses.

The application combines retrieval-based AI methods with a clean, responsive web interface to provide relevant legal information efficiently.

## Live Demo

Live Application:  
https://nyaya-ai-five.vercel.app/

## Features

- Semantic search over Indian legal case data
- AI-assisted chat interface for legal queries
- Retrieval-Augmented Generation (RAG) using FAISS
- Modular and scalable full-stack architecture
- Secure configuration using environment variables

## Technology Stack

**Backend / AI**
