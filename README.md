Nyaya AI

Nyaya AI is an AI-powered legal assistance platform designed to help users explore and understand Indian legal cases through intelligent search and contextual responses.

The project focuses on improving accessibility to legal information by combining semantic search with modern AI techniques in a user-friendly web application.

⸻

Overview

Nyaya AI enables users to query legal case data and receive relevant, context-aware information using retrieval-based AI methods. The system is designed to handle large legal text datasets efficiently while providing a clean and responsive user experience.

⸻

Key Features
	•	Semantic search over Indian legal case data
	•	AI-assisted chat interface for legal queries
	•	Retrieval-Augmented Generation (RAG) using FAISS
	•	Scalable and modular full-stack architecture
	•	Secure configuration using environment variables
